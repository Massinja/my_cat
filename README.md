# Welcome to My Cat
***

## Description
A simplistic implementation of cat command in Linux.  
Programm concatenates files and prints on the standard output 

## Installation
Run command "make" in the terminal. It will build the target file "my_cat"
```
make
```
## Usage
```
./my_cat file1 file2
```
